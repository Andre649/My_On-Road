# My_On-Road
# Roadmap de Transição: Sales/CRM para Product Management
**Autor:** Bucanero
**Objetivo:** Consolidar a experiência em relacionamento com o cliente com competências técnicas de Gestão de Produto.

---

## Visão Geral
Este repositório documenta minha jornada de transição de carreira, fundamentada nos princípios de Marty Cagan (Inspired) e Eric Ries (Lean Startup). O foco é o desenvolvimento de habilidades para mitigar riscos de valor, viabilidade, usabilidade e factibilidade.

## O Roadmap de Desenvolvimento

### FASE 1: Fundamentos e Product Thinking
- Leitura: Inspired (Marty Cagan).
- Estudo: O Triângulo de PM (UX, Tecnologia e Negócios).
- Entregável: Artigo sobre a transição de mentalidade de Entrega (Output) para Resultado (Outcome).

### FASE 2: Discovery e Validação de Hipóteses
- Framework: Jobs To Be Done (JTBD).
- Prática: Mapeamento de Jornada do Usuário aplicada ao setor automotivo/premium.
- Entregável: Estudo de caso sobre a dor do cliente e validação de solução.

### FASE 3: Dados e Execução Ágil
- Estudo: North Star Metric vs. Métricas de Vaidade.
- Ferramenta: Priorização usando o framework RICE (Reach, Impact, Confidence, Effort).
- Entregável: Definição de métricas de sucesso para um MVP de serviço.
